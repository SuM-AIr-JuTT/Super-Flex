# Super Flex

Super Flex is a revised version of Jugaadu-Flex Chrome extension designed for the FlexStudent portal. It provides useful student tools with a clean premium interface, customizable themes, and quick automation features.

## Features

- Fix Grand Total marks
- Edit marks
- GPA calculator
- Fees tool
- Dark mode
- Feedback automation
- 14 built-in themes
- Cyberpunk, anime, glass, black, neon, and light UI styles
- Clean popup interface
- Custom Super Flex logo

Installation
Download or clone this repository.
Open Chrome or a Chromium-based browser.
Go to:
chrome://extensions/
Turn on Developer mode.
Click Load unpacked.
Select the Super Flex project folder.
Open the FlexStudent website and use the extension.
